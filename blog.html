• Discuss the scope of var, let, and const?

    • => var: In JavaScript, variables declared with the var keyword have a function-level or global scope, depending on where they are declared.If you declare a var variable inside a function, it is only accessible within that function.
    => let: The let keyword is used to declare variables in JavaScript with block scope. Variables declared with let have block-level scope, which means they are limited to the block in which they are defined.
    => const: The const keyword is used to declare variables with block scope which means that they are limited to the block (enclosed by curly braces {}) in which they are defined. This includes functions, loops, and any other code blocks.

  • What do you mean by REST API?

    • => Rest API: A REST API (Representational State Transfer Application Programming Interface) is a set of rules and conventions for designing web services that allow different software applications to communicate with each other over the internet. REST APIs use standard HTTP methods like GET, POST, PUT, and DELETE to perform actions on resources, such as retrieving, creating, updating, or deleting data. They are designed to be simple, scalable, and stateless, making them a popular choice for building web services and enabling data exchange between clients (e.g., web browsers, mobile apps) and servers.
